Description:
This folder contains Jupyter Notebook files used throughout the research process. The notebooks include data preprocessing, time series modeling, model evaluation, and result visualization.

List of contents:
- Data cleaning and transformation steps
- Exploratory data analysis (EDA)
- SARIMA model implementation for forecasting
- Model evaluation using MAPE, MAE, and RMSE
- Visualization of actual vs predicted passenger numbers

Tools and libraries used:
- Python (Jupyter Notebook)
- pandas, matplotlib, statsmodels, seaborn

Note:
These notebooks are designed to be run in sequence. Make sure the required datasets are located in the `dataset/` folder.
