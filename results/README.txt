Description:
This folder contains the output results from the SARIMA time series forecasting model. The predictions focus on estimating the number of airplane passengers in Indonesia for the year 2025, based on historical data from 2006 to 2024.

Contents:
- Forecasted values for domestic and international passenger traffic
- Monthly prediction results in Excel format
- Performance evaluation metrics (MAPE, MAE, RMSE)

Usage:
These results can be used for evaluating model accuracy and comparing predicted passenger volumes against actual values (when available).

Note:
The forecasts are based on the best SARIMA models selected for each airport. The original datasets are located in the `dataset/` folder, and modeling scripts are provided in the `notebooks/` folder.
