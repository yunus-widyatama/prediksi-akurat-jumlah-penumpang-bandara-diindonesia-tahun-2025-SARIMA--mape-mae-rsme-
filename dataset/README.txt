Description:
This folder contains a collection of historical air passenger data from 2006 to 2024, covering four major airports in Indonesia: Soekarno-Hatta (CGK), Juanda (SUB), Ngurah Rai (DPS), and Kualanamu (KNO).

Data format:
- The dataset is provided in Excel (.xlsx) format with monthly time series entries.
- Each row represents the number of domestic and international passengers for a specific month.
- The data has been cleaned and structured for time series forecasting purposes.

Column structure (example):
- Date
- Airport Name
- Domestic Passengers
- International Passengers
